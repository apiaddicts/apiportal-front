Referencia de la API

Puedes encontrar toda la documentación actualizada en nuestro API Explorer

Desde el API Explorer podrás realizar pruebas de llamadas a los diferentes endpoints pulsando el botón

Deberás informar:

  • authorization : el valor del access_token precedido del literal "Bearer "
  • api-version: 1.0
  • Ocp-Apim-Subscription-Key: tu clave de suscripción del portal a3developer. Si has hecho ya login en el portal developer, se informará el valor automáticamente.
Atención

Si no conoces tu clave de suscripción o cómo obtener el cliente OAuth, consulta el apartado cómo empezar de esta documentación.
